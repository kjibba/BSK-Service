# Backend (Flask) – Arkivert

Denne mappen er arkivert. Operativ backend er i `../backend-nodejs/`.

- Ikke deploy eller start denne koden i nye miljøer.
- Nye API-endringer og funksjoner skal implementeres i `backend-nodejs/`.
- Eventuelle data- og skjemaendringer håndteres via TypeORM-migrasjoner i Node-prosjektet.

Se rot-`README.md` og `.github/copilot-instructions.md` for gjeldende instruksjoner.
